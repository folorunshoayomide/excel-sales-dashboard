📊 Sales Data Analysis Dashboard (Excel)

📌 Project Overview
This project focuses on analyzing a retail sales dataset to uncover key business insights and present them through an interactive Excel dashboard.

What makes this project unique is the iterative improvement process — the dashboard was initially created, reviewed by  Datacamp Africa Community, and then refined based on feedback to produce a more effective and professional version.

🎯 Objective
To analyze sales performance across regions, product categories, and time, and present insights through a clear and interactive dashboard.

🛠 Tools & Skills Used
- Microsoft Excel 
- Data Cleaning 
- Pivot Tables 
- Pivot Charts 
- Slicers 
- Dashboard Design 
- Data Visualization 

📂 Dataset
Superstore Sales Dataset (sourced from Kaggle)

🔍 Data Cleaning & Preparation
The dataset was cleaned and prepared before analysis:
- Removed duplicate records 
- Handled missing and inconsistent values 
- Standardized data formats (especially date fields) 
- Created helper columns (e.g., Month for trend analysis) 

📈 Analysis Performed
The following analyses were carried out:
- Sales performance by region 
- Sales distribution by product category 
- Monthly sales trend analysis 
- Identification of top-performing products 

Pivot tables and charts were used extensively to summarize and explore the data.

📊 Dashboard Development Process

🔹 Initial Dashboard
The first version of the dashboard was created using:
- Pivot tables 
- Charts 
- Basic layout design 

This version focused on presenting key metrics but had areas for improvement in clarity, structure, and visual design.

🔹 Feedback & Improvement
The initial dashboard was shared with the DataCamp Community Africa group, where it received constructive feedback from experienced members.

The feedback highlighted:
- Layout and design improvements 
- Better organization of visuals 
- Clearer presentation of insights 

🔹 Final Dashboard (Improved Version)
Based on the feedback, the dashboard was redesigned to:
- Improve layout and visual hierarchy 
- Enhance readability and clarity 
- Create a more professional and user-friendly interface 

📌 Both versions are included in the Excel file:
- Sheet 1: Initial Dashboard 
- Sheet 2: Corrected Dashboard 

 📊 Dashboard Features
- Total Sales overview 
- Sales by Region 
- Sales by Category 
- Monthly Sales Trend 
- Top Products by Sales 
- Interactive filters (Slicers) 

📊 Key Insights
- Consumer segment leads sales 
- Technology is the top-performing product category 
- Sales peak towards the end of the year 
- A small number of products contribute significantly to total revenue
